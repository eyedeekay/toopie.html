toopie.html I2P standalone monitoring panel

This is a very simple monitoring panel for interacting with I2P and monitoring I2P statistics. It can stop, or restart an I2P router, as well as provide concise output about I2P router status from I2PControl. It depends on the presence of an I2P router implementing the i2pcontrol jsonrpc interface.
